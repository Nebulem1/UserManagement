# User Management API

A production-style FastAPI backend for secure user authentication, profile management, and role-based access control.

## Technologies Used
- Python 3.12+
- FastAPI & Uvicorn
- SQLAlchemy & SQLite
- Alembic
- Pydantic
- Pytest
- Passlib/Bcrypt
- PyJWT

## Setup Instructions

### Local Development (Virtual Env)

1. **Create and Activate Virtual Environment**
   ```bash
   python -m venv venv
   # Windows
   .\venv\Scripts\Activate.ps1
   # macOS/Linux
   source venv/bin/activate
   ```

2. **Install Dependencies**
   ```bash
   pip install -r requirements.txt
   # Install dev tools if needed via pyproject.toml
   pip install -e .[dev]
   ```

3. **Configure Environment**
   Copy the example environment into a real `.env` file.
   ```bash
   cp .env.example .env
   ```

4. **Initialize Database**
   Since Alembic is initialized, run migrations to generate your `.db` file.
   ```bash
   alembic upgrade head
   ```

5. **Run the Application**
   ```bash
   uvicorn app.main:app --reload
   ```

   Visit `http://localhost:8000/docs` to see the autogenerated Swagger UI API Reference.

### Running with Docker

Run the entire application out-of-the-box using the provided `docker-compose.yml`:
```bash
docker-compose up --build
```
This binds to `8000` automatically.

## Testing

Run tests by leveraging Pytest:
```bash
pytest
```
The testing environment will autogenerate a mock SQLite DB, execute integrations, and drop the tables after execution.
